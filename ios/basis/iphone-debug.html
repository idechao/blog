# iphone无证书真机调试

当前环境：Xcode Version 12.5.1 (12E507)

# 创建证书

我为了测试流程，重新注册了apple id，在xcode中登录，选择 Manage Certificates

点击+号，新建 Apple Development

# 工程配置

Team选择刚新加的Personal Team，这时候，Build Settings里的Code Signing Identity自动变成Applee Development，就用这个就好的

需改Bundle Identifier，这里主要不能和别人一样的，所以如果是本地测试，那就长一点好了

# 信任开发者

配置好直接运行，会有个提示，这时候需要到手机描述文件里，信任证书，然后在重新运行。
