# Electron Security Warning

运行报如下警告：

Electron Security Warning (Insecure Content-Security-Policy) This renderer process has either no Content Security
1

可以直接在main文件上面，添加个忽略：

process.env['ELECTRON_DISABLE_SECURITY_WARNINGS'] = 'true'
